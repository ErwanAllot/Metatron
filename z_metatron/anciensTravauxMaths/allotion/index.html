Sur futura-sciences, un forumeur du nom de Seirios m'a donné une autre formule pour l'aire du tricarré (A).
A = 1/2 ( (a-1) * (b-1) + PGCD(a;b) -1 )
Ma façon de calculer permet dans certains cas d'aller plus vite, généralement pour des nombres assez petits ou dans certains autres cas (j'avais fait une étude en algo pour comparer les 2 méthodes visible sur github).
